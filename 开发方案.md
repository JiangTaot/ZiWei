## 五、排盘算法验证

15 个 engine 文件 + 10 个枚举文件经 `diff` 对比，与 YunDao 参考实现**完全一致**（仅 package 名不同）。  
12 步排盘流水线顺序一致，已与文墨天机 App 对齐。

**关键修复**：前端发送 地支 index (0-11) → 后端转换为 solar hour (index*2) → 引擎正确计算八字/五行局。

## 七、开发命令

```bash
# 后端
cd ziwei-server
mvn spring-boot:run          # 启动 (端口 48121)

# 前端
cd ziwei-app
npm run build:mp-weixin       # 编译到 dist/build/mp-weixin/
# 微信开发者工具 → 打开 dist/build/mp-weixin/

# 代码质量
mvn clean compile              # 后端编译验证
npx uni build -p mp-weixin     # 前端编译验证
```
